# Types of SQL Commands

# 1. DDL
    1. CREATE 
    2. ALTER TABLE  
    3. DROP ->  Delete Table DB , view  
    4. Truncate -> Remove all tuples from the table  
    5. RENAME -> Rename DB name, table name, column name

# 2. DRL/DQL
    1. SELECT

# 3. DML
    1.INSERT
    2.UPDATE
    3.DELETE

# 4. DCL
    1. GRANT 
    2. REVOKE

# 5. TCL
    1. START TRANSACTION
    2. COMMIT
    3. ROLLBACK
    4. SAVEPOINT
